# BasicExtensions

For implement library:
  
  - add dependency to build.gradle (project lvl)
  
  allprojects {
  	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
   }
  
  
  - add dependency to build.gradle (app lvl)
  
  dependencies {
	        implementation 'com.github.Divinkas:BasicExtensions:1.0.1'
	}
